# 🔗 Zaid Khaza'leh — Link Hub

A modern, RTL Arabic personal link-in-bio page with an **IoT-themed animated background** (network of sensors/devices/gateways with data packets travelling along the connections).

## ✨ Features

- 🎨 Glassmorphism design with animated gradient background
- 🌐 IoT network visualization — pulsing nodes, data packets, and WiFi rings
- 📱 Fully responsive (mobile-first)
- 🌍 RTL Arabic with Tajawal + Cairo fonts
- 🔗 Brand-colored social buttons (Instagram, Telegram, Snapchat, GitHub, JUST)
- 📻 Live Quran & Prophet's Mosque radio links
- 📤 Floating share modal (WhatsApp / Telegram / X / Snapchat / Copy)
- ♿ Accessible (ARIA labels, keyboard navigation, `prefers-reduced-motion`)
- ⚡ Zero build step — pure HTML/CSS/JS

## 🚀 Stack

- **HTML5** (semantic, accessible)
- **CSS3** (custom properties, glassmorphism, animations)
- **Vanilla JS** (canvas particles, share modal, no dependencies)
- **FontAwesome 6** (icons)
- **Google Fonts** (Tajawal + Cairo)
